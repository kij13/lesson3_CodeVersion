Highlight #1 Version Control

Version control can be used to manage many versions of computer files and programs. It can allow users to trach changes that are made to files. This type of version control would mostly be used for software development. Version control software requires that all files are saved in a central location, and this location is known as the repository.

Highlight #2 GIT

GIT is a free and open source distributed version control system. It is designed to handle any project, small or large with great speed and preciseness. GIT is also extremely easy to learn.

Highlight #3 Local Repository

A local repository is a type of repository that resides on a local machine that of the individual user. There are 2 ways that a local repository can be created. The first way is to create a completely brand new repository or to clone an existing remote repository onto ones local machine.
